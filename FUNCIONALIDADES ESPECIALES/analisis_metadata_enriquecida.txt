## ⚠️ LIMITACIONES ACTUALES (Por entorno)

### 1. **Tags con cobertura 83.2%**
**Problema**: No todos los videos tienen tags definidos por YouTube.

**Estado actual**:
- Cobertura: 2500/3004 archivos con tags
- Causa: Creadores no siempre añaden tags manualmente

**Solución**: El MetadataEnricher puede activar modo activo/híbrido para estos casos usando YouTube API

### 2. **Descripción truncada a 2000 chars**
**Problema**: Descripciones largas se cortan para evitar bloating en DB.

**Estado actual**:
- Campo `description`: 2000 chars max
- Campo `description_full`:保留了 descripción completa en metadata_json

**Solución**: Búsqueda en keywords compensa, acceso a metadata_json para texto completo

### 3. **YouTube API requiere setup adicional**
**Problema**: Para enriquecimiento activo se necesita Google Cloud Console + API key.

**Estado actual**:
- Modo default: `passive` (sin API key necesaria)
- Solo activa si: `enrichment_mode != 'passive'` Y API key configurada

**Solución**: Documentar en README cómo obtener API key si se necesita modo activo
