## 📊 **CAMPO NOMBRE DEL CANAL - ARQUITECTURA ELITE CORPORATE**

Transformar el campo "Nombre" de un input manual básico a un **Sistema Híbrido de Denominación Inteligente** que sugiere automáticamente nombres descriptivos basados en el contenido del canal, valida coherencia con la URL, detecta el nicho KDP específico y organiza automáticamente en la estructura de 18 categorías, reduciendo el tiempo de configuración de 2 minutos a 10 segundos por canal.

El usuario solo confirma o edita ligeramente. **Tiempo ahorrado: 30 segundos → 3 segundos por canal.**

## 📦 **LISTA 1: 60 MÓDULOS SIN IA (Infraestructura Determinista & Validación de Nombres)**

### **Pilar 1: Validación y Normalización de Nombres (1-12)**

1.  **Validador de Longitud Mínima/Máxima** — Verifica que el nombre tenga entre 3-100 caracteres. *Caso de uso real:* Usuario escribe "A" → sistema alerta "❌ Nombre demasiado corto (mínimo 3 caracteres)"; escribe 150 caracteres → "❌ Nombre demasiado largo (máximo 100 caracteres)".

2.  **Validador de Caracteres Especiales** — Permite solo caracteres alfanuméricos, espacios, guiones y emojis básicos. *Caso de uso:* Usuario escribe "Canal@KDP#2026!" → sistema limpia automáticamente a "Canal KDP 2026" o alerta "⚠️ Caracteres especiales no permitidos".

3.  **Normalizador de Mayúsculas/MINÚSCULAS** — Convierte automáticamente a Title Case (Primera Letra Mayúscula). *Caso de uso:* Usuario escribe "pablo alanzahlut - kdp expert" → sistema convierte automáticamente a "Pablo Alanzahlut - KDP Expert".

4.  **Detector de Nombres Duplicados** — Verifica si el nombre ya existe en la base de datos de canales monitoreados. *Caso de uso:* Usuario intenta añadir "Pablo KDP" pero ya existe → sistema alerta "⚠️ Ya existe un canal con este nombre. ¿Usar 'Pablo KDP (2)'?".

5.  **Validador de Coherencia URL-Nombre** — Verifica que el nombre guarde relación con la URL del canal. *Caso de uso:* URL es "youtube.com/@pablokdp" pero usuario escribe "Canal de Cocina" → sistema alerta "⚠️ El nombre no parece coincidir con la URL del canal".

6.  **Eliminador Automático de Espacios Extra** — Elimina espacios al inicio, final y duplicados en medio. *Caso de uso:* Usuario escribe "  Pablo   KDP  " → sistema limpia automáticamente a "Pablo KDP".

7.  **Validador de Idioma del Nombre** — Detecta si el nombre está en español, inglés u otro idioma. *Caso de uso:* Sistema detecta "🌐 Idioma detectado: Inglés" → alerta si usuario solo quiere canales en español.

8.  **Detector de Nombres Genéricos** — Identifica nombres como "Canal1", "Test", "Nuevo Canal". *Caso de uso:* Usuario escribe "Canal1" → sistema sugiere "✏️ ¿Usar nombre descriptivo? Ej: 'Pablo KDP - Amazon Ads'".

9.  **Validador de Profanidades/Spam** — Filtra palabras inapropiadas o spam en el nombre. *Caso de uso:* Nombre contiene "FREE MONEY!!!" → sistema alerta "⚠️ Nombre contiene posible spam o contenido inapropiado".

10. **Autoguardado de Últimos Nombres Usados** — Guarda historial de los últimos 10 nombres personalizados. *Caso de uso:* Usuario empieza a escribir "Pab" → sistema sugiere autocompletar con "Pablo Alanzahlut - KDP Expert" del historial.

11. **Validador de Unicidad por Categoría** — Verifica que no haya dos canales con mismo nombre en misma categoría. *Caso de uso:* Usuario tiene "Pablo KDP" en categoría "Amazon Ads" e intenta añadir otro "Pablo KDP" en misma categoría → alerta "⚠️ Nombre duplicado en esta categoría".

12. **Normalizador de Emojis** — Permite solo emojis específicos relacionados con KDP (📚, , 🎯, etc.). *Caso de uso:* Usuario escribe "Canal KDP 📚" → sistema valida emoji permitido; escribe "Canal 💰💰" → sistema alerta "⚠️ Emoji no recomendado para nombres profesionales".

### **Pilar 2: Autocompletado y Sugerencias Automáticas (13-24)**

13. **Extracción Automática desde YouTube API** — Obtiene el nombre oficial del canal desde la API de YouTube. *Caso de uso real:* Usuario pega URL "youtube.com/@pablokdp" → sistema autocompleta automáticamente campo Nombre con "Pablo Alanzahlut - KDP Expert" desde YouTube.

14. **Sugerencia Basada en Handle (@)** — Extrae nombre del handle de YouTube si no hay nombre personalizado. *Caso de uso:* Canal tiene handle "@pablokdp" pero no ha configurado nombre → sistema sugiere "Pablo KDP" basado en el handle.

15. **Autocompletado desde Canales Similares** — Sugiere nombres basados en patrones de canales similares ya añadidos. *Caso de uso:* Usuario tiene "Pablo KDP - Amazon Ads", "María KDP - Low Content" → al añadir nuevo canal de María, sugiere "María KDP - [Categoría]".

16. **Detección de Patrón de Nomenclatura** — Aprende el patrón que usa el usuario (ej. "Nombre - Categoría"). *Caso de uso:* Usuario siempre usa formato "Nombre - Categoría" → sistema sugiere automáticamente ese formato para nuevos canales.

17. **Sugerencia desde Metadatos del Canal** — Usa descripción del canal para sugerir nombre más descriptivo. *Caso de uso:* Descripción dice "Experto en Amazon KDP y Low Content Books" → sistema sugiere "Pablo KDP - Low Content Expert".

18. **Historial de Ediciones de Nombre** — Guarda versiones anteriores del nombre por si el usuario quiere revertir. *Caso de uso:* Usuario cambia "Pablo KDP" a "Pablo Amazon Expert" pero se arrepiente → puede revertir a versión anterior.

19. **Sugerencia Basada en Videos Recientes** — Analiza títulos de últimos videos para sugerir especialización. *Caso de uso:* Últimos 5 videos son sobre "Amazon Ads" → sistema sugiere "Pablo KDP - Amazon Ads Specialist".

20. **Detección de Sinónimos de Categoría** — Reconoce que "PPC", "Ads", "Publicidad" son lo mismo para Amazon KDP. *Caso de uso:* Canal se especializa en "PPC" → sistema sugiere nombre con "Amazon Ads" (término estandarizado en las 18 categorías).

21. **Autocompletado Inteligente por Tecla Tab** — Permite navegar entre sugerencias con tecla Tab. *Caso de uso:* Usuario escribe "Pab" → pulsa Tab → sistema completa "Pablo Alanzahlut - KDP Expert".

22. **Sugerencia desde Base de Conocimiento** — Si el canal ya está mencionado en KB, sugiere ese nombre. *Caso de uso:* Canal ya fue mencionado en transcripciones como "Pablo Alanzahlut (KDP Expert)" → sistema sugiere ese nombre exacto.

23. **Detección de Abreviaturas Comunes** — Reconoce que "KDP" = "Kindle Direct Publishing". *Caso de uso:* Usuario escribe "Kindle Direct Publishing Expert" → sistema sugiere versión corta "KDP Expert" (más común en la comunidad).

24. **Sugerencia Basada en Frecuencia de Búsqueda** — Si el usuario busca frecuentemente este canal, sugiere nombre prominente. *Caso de uso:* Usuario busca "Pablo KDP" 20 veces este mes → sistema sugiere nombre prominente y fácil de recordar.

### **Pilar 3: Organización y Categorización Automática (25-36)**

25. **Asignación Automática a 18 Categorías KDP** — Detecta automáticamente a cuál de las 18 categorías maestras pertenece. *Caso de uso real:* Nombre sugiere "Amazon Ads" → sistema asigna automáticamente a categoría #4 "Amazon Ads (PPC)".

26. **Detección de Sub-Categorías** — Identifica sub-nichos dentro de categorías principales. *Caso de uso:* Nombre es "Pablo KDP - Coloring Books" → sistema asigna a "Low-Content > Coloring Books".

27. **Validación de Coherencia Categoría-Nombre** — Verifica que el nombre sea coherente con la categoría asignada. *Caso de uso:* Nombre dice "Low Content" pero categoría es "Amazon Ads" → sistema alerta "⚠️ Inconsistencia detectada entre nombre y categoría".

28. **Sugerencia de Tags Basados en Nombre** — Extrae keywords del nombre para usar como tags. *Caso de uso:* Nombre es "Pablo KDP - Amazon Ads Expert" → sistema extrae tags automáticos: #AmazonAds #PPC #Marketing.

29. **Mapeo de Nombre a Rol SOE** — Asocia el canal con uno de los 37+1 roles GEM basado en el nombre. *Caso de uso:* Nombre contiene "SEO" → sistema sugiere asignar a Rol #6 "Especialista SEO Amazon (A9)".

30. **Detección de Nivel de Experto** — Identifica si el nombre sugiere nivel principiante, intermedio o avanzado. *Caso de uso:* Nombre es "KDP para Principiantes" → sistema etiqueta como "Nivel: Principiante".

31. **Agrupación Automática por Patrón de Nombre** — Agrupa canales con nombres similares. *Caso de uso:* Usuario tiene "Pablo KDP - Ads", "Pablo KDP - SEO", "Pablo KDP - Legalidad" → sistema los agrupa visualmente como "Serie Pablo KDP".

32. **Validación de Consistencia de Marca** — Verifica si el nombre mantiene consistencia de branding con otros canales del mismo creador. *Caso de uso:* Usuario tiene "Pablo KDP", "María Marketing" → añade "Pablo Amazon" → sistema sugiere "✏️ ¿Usar 'Pablo KDP - Amazon' para mantener consistencia?".

33. **Detección de Canales de Competencia** — Identifica si el nombre sugiere que es competencia directa. *Caso de uso:* Nombre contiene "Competencia" o es similar a canales competidores → sistema etiqueta como "⚠️ Competencia".

34. **Asignación de Prioridad Basada en Nombre** — Asigna prioridad alta si el nombre contiene palabras clave importantes. *Caso de uso:* Nombre es "Amazon Ads EXPERT 2026" → sistema asigna prioridad "Alta" automáticamente.

35. **Detección de Actualidad en Nombre** — Verifica si el nombre contiene año actual (2026) o está desactualizado. *Caso de uso:* Nombre es "KDP Expert 2023" → sistema alerta "⚠️ Nombre desactualizado (2023). ¿Actualizar a 2026?".

36. **Mapeo de Nombre a Manual KDP** — Sugiere a qué manual (Legalidad, Fórmulas, Matriz) pertenece el contenido del canal basado en el nombre. *Caso de uso:* Nombre contiene "Legal" o "TOS" → sistema sugiere integrar en "MANUAL_LEGALIDAD.md".
