# 🔐 SECURITY RULES

- Validación de inputs obligatoria
- No exposición de datos sensibles
- Manejo de errores seguro
- Validación de IA: no enviar binarios/imágenes a modelos de texto
- Verificar tipo de contenido antes de llamadas a APIs
